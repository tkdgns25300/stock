# InvestStats - 투자 통계 정보 조회 서비스

InvestStats는 주식, 채권 등 투자를 위해 필요한 다양한 자료들을 손쉽게 조회할 수 있는 통계 정보 조회 서비스입니다. 이 서비스는 Nest.js를 기반으로 구축되었으며, 주식 시장과 채권 시장의 다양한 통계 정보를 API 형태로 제공합니다.

## 기능

### 지수

KRX 시리즈 일별시세정보 : KRX 시리즈 지수의 시세정보 제공<br>
KOSPI 시리즈 일별시세정보 : KOSPI 시리즈 지수의 시세정보 제공<br>
KOSDAQ 시리즈 일별시세정보 : KOSDAQ 시리즈 지수의 시세정보 제공<br>
채권지수 시세정보 : 채권지수의 시세정보 제공<br>
파생상품지수 시세정보 : 파생상품지수의 시세정보를 제공<br>

### 주식

유가증권 일별매매정보 : 유가증권시장에 상장되어 있는 주권의 매매정보 제공<br>
코스닥 일별매매정보 : 코스닥시장에 상장되어 있는 주권의 매매정보 제공<br>
코넥스 일별매매정보 : 코넥스시장에 상장되어 있는 주권의 매매정보 제공<br>
신주인수권증권 일별매매정보 : 유가증권/코스닥시장에 상장되어 있는 신주인수권증권의 매매정보 제공<br>
신주인수권증서 일별매매정보 : 유가증권/코스닥시장에 상장되어 있는 신주인수권증서의 매매정보 제공<br>
유가증권 종목기본정보 : 유가증권 종목기본정보<br>
코스닥 종목기본정보 : 코스닥 종목기본정보<br>
코넥스 종목기본정보 : 코넥스 종목기본정보<br>

### 증권상품

ETF 일별매매정보 : ETF(상장지수펀드)의 매매정보 제공<br>
ETN 일별매매정보 : ETN(상장지수증권)의 매매정보 제공<br>
ELW 일별매매정보 : ELW(주식위런트증권)의 매매정보 제공<br>

### 채권

국채전문유통시장 일별매매정보 : 국채전문유통시장에 상장되어있는 채권의 매매정보 제공<br>
일반채권시장 일별매매정보 : 일반채권시장에 상장되어있는 채권의 매매정보 제공<br>
소액채권시장 일별매매정보 : 소액채권시장에 상장되어있는 채권의 매매정보 제공<br>

### 파생상품

선물 일별매매정보 (주식선물外) : 파생상품시장의 선물 중 주식선물을 제외한 선물의 매매정보 제공<br>
주식선물(유가) 일별매매정보 : 파생상품시장의 주식선물 중 기초자산이 유가증권시장에 속하는 주식선물의 거래정보 제공<br>
주식선물(코스닥) 일별매매정보 : 파생상품시장의 주식선물 중 기초자산이 코스닥시장에 속하는 주식선물의 거래정보 제공<br>
옵션 일별매매정보 (주식옵션外) : 파생상품시장의 옵션 중 주식옵션을 제외한 옵션의 매매정보 제공<br>
주식옵션(유가) 일별매매정보 : 파생상품시장의 주식옵션 중 기초자산이 유가증권시장에 속하는 주식옵션의 거래정보 제공<br>
주식옵션(코스닥) 일별매매정보 : 파생상품시장의 주식옵션 중 기초자산이 코스닥시장에 속하는 주식옵션의 거래정보 제공<br>

### 일반상품

석유시장 일별매매정보 : KRX 석유시장의 매매정보 제공<br>
금시장 일별매매정보 : KRX 금시장 매매정보 제공<br>
배출권 시장 일별매매정보 : KRX 탄소배출권 시장의 매매정보 제공<br>

### ESG

사회책임투자채권정보 : 사회책임투자채권 정보를 제공 (‘19년 데이터부터 제공)<br>
